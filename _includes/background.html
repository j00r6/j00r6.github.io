저는 비전공자입니다. 음악을 좋아해서 보컬을 전공하였고, 가수가 되고싶었지만 현실을 직시하여
Sound Engineer 로서 주식회사 다락컴퍼니 에 입사하여 근무했습니다.
이런 제가 29년 동안 살아온 커리어를 없애고 새롭게 IT 계열로 입사하게된 계기가 있습니다.

다락컴퍼니에서 제 역할은 다양했지만 그중 하나인 LMS 컨텐츠 제작 에도 참여했었습니다.
해당 프로젝트에서 제 역할은 제작이된 영상 컨텐츠를 웹에 게시하고 배포하는 역할을 맡았습니다.
그 과정중에 JavaScript 언어로 작성된 Backend 파일들을 주로 수정했습니다.

처음하는 업무이고 업무를 가르쳐줄 사수의 부재로 초반엔 많이 헤맸지만
직원들과 같이 협력하여 무사히 프로젝트를 잘 마칠수 있었습니다.

그렇게 해당 프로젝트가 종료되고 코딩에 흥미를 느껴 JavaScript 언어를 독학했지만
좀더 전문적으로 학습해보고 싶다는 생각이 들어서 코드스테이츠 부트캠프 백엔드 과정에 등록했습니다.
그렇게 해당 과정을 수료하고 IT 업계로의 진출을 준비하게 됐습니다.
